# Changelog

Все значимые изменения в проекте PayrollSystem документируются в этом файле.

## [1.8.2] - 2026-04-06

### 🔄 Изменено

#### Payroll Service - Логика удержаний
- **Fixed Rate (8€/h)** - теперь удержания ПРИМЕНЯЮТСЯ (housing + utilities вычитаются из gross)
- **Custom Rate** - теперь удержания НЕ применяются (net = gross, без вычетов)

### 📝 Описание
Изменена логика расчета зарплаты: удержания за коммунальные услуги и аренду теперь применяются только к фиксированной ставке 8€/час, а при использовании кастомной ставки сотрудника удержания не применяются.

## [1.8.1] - 2026-03-03

### 🐛 Исправлено

#### Database
- **Метод `add_employee()`** - добавлена поддержка банковских реквизитов (bank, iban, bic)
- **Новый метод `update_employee()`** - создан метод для обновления всех полей сотрудника

#### UI - Employees Tab
- **Кнопка Add** - исправлена работа с банковскими реквизитами
- **Кнопка Update** - добавлена валидация имени и автоматическая очистка формы после обновления
- **Улучшенная валидация** - проверка на пустое имя при обновлении сотрудника

### 🔧 Техническое
- Синхронизация методов БД с UI компонентами
- Улучшенная обработка банковских данных

## [1.8.0] - 2026-02-18

### ✨ Добавлено

#### Payroll Tab - Профессиональная модернизация
- **3-зонный layout** - Header / Content / Summary для структурированного интерфейса
- **Горизонтальная панель управления** - Employee / Period / Generate в одной строке
- **Улучшенная таблица часов**:
  - Zebra-стиль (чередующиеся цвета строк)
  - Подсветка выходных дней (красный фон)
  - Итоговая строка с общим количеством часов
  - Scrollbar для длинных периодов
- **Карточка Rate Type** - радиокнопки для выбора Fixed/Custom rate
- **Карточка Deductions** - поля Utilities/Rental с preview расчёта
- **Горизонтальные PDF кнопки** - Preview / Save / Print в одной строке
- **Summary Panel** - панель итогов с Total Hours / Gross / Deductions / NET
- **Автоматический пересчёт** - мгновенное обновление NET при изменении параметров

### 🎨 Изменено
- Полная реорганизация UI вкладки Payroll
- Профессиональный дизайн как в бухгалтерских системах
- Улучшенная визуальная иерархия элементов
- Цветовое кодирование (синий - информация, зелёный - доход, красный - вычеты)

### 🔧 Техническое
- Интеграция с `payroll_service` для расчётов
- Reactive updates через StringVar traces
- Оптимизированная структура кода

## [1.7.0] - 2026-02-18

### ✨ Добавлено

#### Интерфейс
- **Zebra-стиль для таблиц** - чередующиеся цвета строк (#f9f9f9 / #ffffff) для лучшей читаемости
- **Поиск сотрудников** - поле поиска с фильтрацией в реальном времени по имени, банку, IBAN и BIC
- **Статус-бар** - отображение пути к базе данных и количества сотрудников
- **Иконки в кнопках** - визуальные индикаторы для действий (➕ Add, ✏️ Update, 🗑️ Delete)
- **Scrollbar для таблицы** - вертикальная прокрутка для больших списков

- **Карточный дизайн форм** - использование Card.TFrame с отступами 20px
- **Горизонтальное расположение кнопок** - три кнопки в ряд вместо вертикального стека
- **Удалена кнопка "Add New"** - упрощённый workflow без лишних действий
- **Улучшенные отступы** - padding 15-20px для всех элементов форм
