# Payroll System

Desktop payroll management system for Windows.

## Features
- Employee management
- Payroll calculation (fixed & custom rates)
- Optional deductions (housing, utilities)
- PDF payroll reports
- Excel export
- SQLite database
- Windows installer (Inno Setup)

## Tech stack
- Python 3.12
- Tkinter
- SQLite
- PyInstaller
- Inno Setup
- pytest
